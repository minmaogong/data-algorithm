数据结构
    一种数据的组织和存储方式
    分类
        按逻辑划分
            线性结构
                除了第一个元素没有前驱元素，最后一个元素没有后继元素，其余每个元素有且只有一个前驱和后继元素
                元素之间的关系是1对1的
            非线性结构
                元素之间具有多个对应关系

        按物理结构划分(数据在计算机内存中的存储结构)
            连续存储:数据在内存中分配的空间地址是连续的

            分散存储:数据在内存中分配的空间地址不是连续的

算法
    解决某一个问题的方法
    分类
        搜索、排序、分治、动态规划、回溯。。。。

算法时间复杂度
    时间复杂度用来描述运行一个算法所需时间资源的多少。
    描述 随着数据规模的不断增长，算法所需要的时间资源的变化趋势

    为了屏蔽计算机硬件差异对评估结果的影响，我们不使用算法运行的绝对运行时间，
    而是使用运行算法时执行的基本指令数量来衡量其所需的时间资源。

    最优时间复杂度
    平均时间复杂度
    最坏时间复杂度  我们主要关注

    一般我们用大O表示法，表示算法的时间复杂度  bigO
        例如：  T(n) = 5n+5        => O(n)
               T(n) = 2n^2+2n-2    => O(n^2)

算法空间复杂度
    描述 随着数据规模的不断增长，算法所需要的内存资源的变化趋势

关于python中的list和array模块下的array
    list:是python内置数据类型

    python和其它编程语言不一样，没有提供内置的数组类型
    要想去使用数组，可以使用array.array()或者numpy模块的ndarray

    数组:
        用来存储相同数据类型的数据结构
        数组是有序的，可以通过下标获取元素
        数组在内存中会分配一篇连续的存储空间
        数组在创建时需要指定长度，并且数组一旦创建，长度就无法改变，如果需要扩容，只能创建一个更大的数组，再将原数据拷贝到新数组
        但是array.array是支持动态扩容的
